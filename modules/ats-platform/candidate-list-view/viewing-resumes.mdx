## Viewing Resumes

This feature provides quick access to candidate resumes directly from the Pipeline View, allowing for efficient review of candidate qualifications and experience.

## Benefits

- Quickly assess candidate suitability without leaving the ATS
- Compare multiple candidates easily by opening resumes in separate tabs
- Save time by having all relevant information accessible in one place

## How to View a Resume

<Steps>
  <Step title="Locate the Candidate">
    Find the candidate in the Pipeline View or list view.
  </Step>
  <Step title="Click Resume Button">
    Click the "Resume" button next to the candidate's name.
  </Step>
  <Step title="Review Resume">
    The resume will open in a new tab or window for your review.
  </Step>
</Steps>

## Resume Viewer Features

Our built-in resume viewer offers several helpful features:

- **Zoom**: Adjust the zoom level for comfortable reading
- **Download**: Save a local copy of the resume for offline review
- **Print**: Send the resume directly to a printer
- **Share**: Generate a temporary link to share the resume with team members

<Tip>
  Use the "Compare" feature to open multiple resumes side-by-side for easier evaluation of candidates.
</Tip>

## Supported File Types

Our system supports various file types for resumes, including:

- PDF (.pdf)
- Microsoft Word (.doc, .docx)
- Rich Text Format (.rtf)
- Plain Text (.txt)

<Warning>
  If you encounter issues viewing a resume, check that it's in one of the supported file formats. For unsupported formats, ask the candidate to resubmit in a supported format.
</Warning>
